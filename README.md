# Liam L. Chemistry A Breakdown & Analysis

An interactive analysis of chemistry questions to understand foundational gaps and build a path forward for Chemistry B and AP Chemistry.

## 📊 Overview

This project provides a comprehensive analysis of chemistry questions covering 10 essential lessons from Chemistry A:
- Scientific Models & Measurements
- Periodic Table
- Ionic Compounds and Metals
- Covalent Compounds
- Complex Structures
- Organic Compounds
- Chemical Reaction Patterns
- Advanced Reaction Patterns
- Reaction Rates and Chemical Equilibrium

## 🎯 Features

- **Interactive Table**: Each question is displayed in an easy-to-read card format
- **Expandable Sections**: Click on any colored section to reveal detailed explanations for:
  - What This Tested (Green) - The specific concept and topic
  - Chem B Impact (Red) - Why this matters for Chemistry B
  - AP Chem Impact (Purple) - Why this matters for AP Chemistry
  - Consequences (Orange) - What happens if the topic is skipped
  - Recommendation (Teal) - Which Chemistry A lesson to study
- **Color-Coded**: Easy visual navigation with distinct colors for each category
- **Responsive Design**: Works on both desktop and mobile devices

## 💡 How to Use

1. Browse through each question card
2. Read the original question and answer choices
3. Click on any of the five colored sections to expand detailed explanations
4. Use the analysis to identify which Chemistry A lessons need review
5. Focus on topics that are highlighted as important for future courses

## 🎨 Color Legend

- **Green (What This Tested)**: Understanding the core concept tested
- **Red (Chem B Impact)**: Connection to Chemistry B curriculum
- **Purple (AP Chem Impact)**: Connection to AP Chemistry curriculum
- **Orange (Consequences)**: Risks of not mastering the concept
- **Teal (Recommendation)**: Specific Chemistry A lesson to review
